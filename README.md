# minecraft-auto-fishing

## Prerequisite

### Install python3
https://www.python.org/downloads/

### Required Library
```
pip3 install pyautogui
pip3 install pillow
```
### Preparing Image

1. Set show subtitle to True
2. Do manual fishing
3. Capture the subtitle "Fish Bobber splashes"
4. Replace "detect-img.png" with the newly captured image


## Running the script
```
python3 fish.py
```
